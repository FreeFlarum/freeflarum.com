## How to run locally:

Instructions for Ubuntu:
```
sudo apt install python3-pip 
sudo pip3 install pipenv
pipenv install
pipenv run mkdocs serve
```
And see your live edits at http://localhost:8000/

## How to deploy:

```
./deploy.sh
```
